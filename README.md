# Felipe Kummer Portfolio RPG

A game-first web portfolio built with Vite, React, TypeScript, and Phaser 3.

## Stack

- Vite
- React
- TypeScript
- Phaser 3
- localStorage for MVP persistence

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`

## Runtime

This repository is pinned to Node `24.14.0` through `.nvmrc`.

## Current Scope

The current foundation includes:

- a thin client-side React shell
- a mounted Phaser game instance
- a boot scene
- an intro scene
- a character select scene
- a world scene placeholder
- asset folder structure for the RPG MVP
